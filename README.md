# ⚖️ AI Legal Contract Risk Intelligence Platform

## 🚀 Overview

An AI-powered LegalTech platform that analyzes contracts, detects compliance risks, and generates explainable legal insights using NLP, RAG, and multi-agent architecture.

---

## 💡 Features

- Contract clause analysis
- Compliance risk detection
- Explainable AI insights
- RAG-powered legal intelligence
- Multi-agent coordination
- Streamlit dashboard

---

## ▶️ Installation

pip install -r requirements.txt

---

## ▶️ Run Project

streamlit run app.py
